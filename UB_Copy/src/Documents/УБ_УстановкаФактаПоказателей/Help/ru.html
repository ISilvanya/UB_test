Документ Установка факта показателей.

Предназначен для ручной установки фактического значения запланированных показателей, у которых реквизит "Вариант расчета значения факта" в схеме сбора данных показателей эффективности равен "Из документа "Установка факта показателя".
Для корректного расчета нарастающего итога важно соблюдать последовательность заполнения факта во времени.

Полноценный контроль соответствия показателя и назначения на данный момент реализован  только на этапе проведения документа. В процессе заполнения он осуществляется лишь частично.
